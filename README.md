# Pinterest Clone

This project is a simple Pinterest-style layout built using HTML and CSS. It displays images in a grid format similar to the Pinterest homepage.

## Features

* Clean and simple UI
* Grid-based image layout
* Basic styling using CSS

## Note

This project is currently optimized for desktop view and may not be fully responsive on mobile devices.
